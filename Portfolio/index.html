How to Play Samurai Adventure Game

2. Movement: Use the arrow keys to move the samurai up, down, left, or right.
3. Avoid Obstacles: Press the up arrow to jump over boxes. If touched, the samurai loses 5% health and teleports to a random location. If samurai touches the screens border it will also teleport to a random location.

Description: This project is a fully functional Flappy Bird game created using JavaScript. It showcases a variety of programming concepts, including object-oriented design, game state management, collision detection, and audio integration. [Skip to 0:27 to view gameplay]

Technologies Used: JavaScript and HTML with the use of sprite and audio assets.

How to Play Flappy Bird Game
1. Start the Game: Press the [SPACE] key on the start screen to begin the game.
2. Movement: Press the [SPACE] key to make the bird flap its wings and rise. Release the [SPACE] key to let the bird fall.
3. Objective: Navigate the bird through gaps in the pipes to avoid collisions. Collect the rings that appear on the screen to increase your score. As you collect more rings and progress through the game the speed increases.
4. Game Over: The game ends if the bird collides with a pipe or the bar/ground. On the game over screen, press [Y] to restart or [N] to exit the game.
